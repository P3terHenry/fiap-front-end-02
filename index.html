O que é Front-end?

O desenvolvimento front-end é a área responsável pela criação da interface com a qual os usuários interagem em um site ou aplicativo web.

Isso inclui tudo o que os usuários veem, clicam ou interagem diretamente em seus navegadores da web ou dispositivos móveis.

Os desenvolvedores front-end usam uma variedade de tecnologias para criar interfaces de usuário atraentes e funcionais que atendam às necessidades dos usuários e atendam aos requisitos do projeto.

Tecnologias utilizadas com Front-end

HTML (Hypertext Markup Language):

  • Estrutura básica de uma página web.
  • Uso de tags para definir elementos da página.
  • Semântica HTML para melhor acessibilidade e SEO.

CSS (Cascading Style Sheets):

  • Estilização de elementos HTML.
  • Seletores CSS e suas aplicações.
  • Layouts responsivos e técnicas de design.

JS (JavaScript):

  • Linguagem de programação utilizada para tornar as páginas web interativas.
  • Manipulação do DOM (Document Object Model).
  • Integração de funcionalidades dinâmicas e reativas.
